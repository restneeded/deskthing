# Wake word models

Put your Picovoice Porcupine `.ppn` files here.

## Lumen (recommended)

1. Create a custom keyword **Lumen** at https://console.picovoice.ai/
2. Download the build for **this host OS** (Windows / macOS / Linux)
3. Save as e.g. `Lumen_en_windows_v3_0_0.ppn` in this folder
4. In Aura settings set **Wake model path** to the full path of that file
   (or a path relative to the DeskThing app working directory)

Do **not** commit `.ppn` files if your license forbids redistribution — they stay on your machine.

See [docs/WAKE_WORD.md](../docs/WAKE_WORD.md).
